## ⚠️ Common Issues & Solutions

### Issue 1: Website shows old version
**Solution:**
- Clear browser cache (Ctrl + Shift + Delete)
- Try incognito mode
- Wait 5-10 minutes for CDN cache

### Issue 2: CSS not loading
**Solution:**
- Check if assets folder uploaded correctly
- Verify file permissions (644 for files, 755 for folders)
- Clear browser cache

### Issue 3: 404 errors on pages
**Solution:**
- Check if .htaccess file is present
- If not, create .htaccess with:
```apache
<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /
  RewriteRule ^index\.html$ - [L]
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteCond %{REQUEST_FILENAME} !-d
  RewriteRule . /index.html [L]
</IfModule>
```

### Issue 4: Sitemap still shows error
**Solution:**
- Wait 24 hours
- Check if sitemap.xml is accessible
- Re-submit in Search Console
- Verify robots.txt allows Googlebot
